# scifio-UView (archived)

> **This repository is no longer maintained.**
> The plugin has moved to [jufiba/LEEMandPEEM](https://github.com/jufiba/LEEMandPEEM), where it is actively developed.

---

This was a SCIFIO plugin for reading images in the Elmitec UKSOFT2000 format (`.dat` extension) used by the U-view 2002 software for low-energy electron microscopy (LEEM) and photoemission microscopy (PEEM).

The migration to [jufiba/LEEMandPEEM](https://github.com/jufiba/LEEMandPEEM) also updated the plugin to the SCIFIO 0.45 API (`DataHandle<Location>`) required by current Fiji releases.
